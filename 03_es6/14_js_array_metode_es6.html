    includes()

    Provera da li JavaSkript niz sadrži traženi element .includes()

    .includes() metod prima traženu vrednost i vraća bulijan u zavisnosti od toga da li Array sadrži prosleđenu vrednost ili ne:

    const mojArray = ["banana", "jabuka", "jagoda"];

    mojArray.includes("banana");
    // Vraća true

    mojArray.includes("cipela");
    // Vraća false

    Pronalaženje člana JavaSkript niza koji zadovoljava zadati uslov .find()

    find prima funkciju kojom je definisan uslov pretrage, i može da primi vrednost this-a, opciono. Prosleđena funkcija prima vrednost trenutnog člana, a opciono može da primi i trenutnu vrednost inexa, i sam array. Metoda proverava svaki član array-a za uslov zadat funkcijom, i vraća prvi član koji zadovoljava uslov. Ukoliko ne pronađe traženi rezltat, vraća undefined.

    const mojArray = [1,6,3];
    mojArray.find(x => !(x%2));

    // Vraća 6 kao prvi paran broj u Array-u

    Kreiranje JavaSkript niza iz stringa ili objekta .from()

    .from() metoda prima neki tip podatka koji nije array, i konvertuje ga u array. Može konvertovati stringove, objekte i sl, ili drugim rečima, može proizvesti array od bilo kog objekta koji ima .length svojstvo ili je iterabilan.

    const mojArray = Array.from("otorinolaringologija");

    // Vraća [ "o", "t", "o", "r", "i", "n", "o", "l", "a", "r", "i", "n", "g", "o", "l", "o", "g", "i", "j", "a" ]

    Metoda obavezno prima jedan argument, koji je objekat koji je potrebno konvertovati u array (kad se kaže objekat, to ne isključuje stringove jer stringovi imaju String() konstruktor, odnosno imaju svoj wraper objekat). Takođe može da primi i dva opciona parametra, mapirajuću funkciju, koja bi se pozivala nad svakim elementom array-a, i vrednost this-a.

    Index elementa JavaSkript niza koji zadovoljava uslov .findIndex()

    Pronalazi i vraća index prvog elementa Array-a koji zadovoljava postavljeni uslov. Ako metoda prođe ceo Array i ne pronađe traženi element, vratiće -1

    Metoda prima do dva parametra, prvi je funkcija po kojoj se vrši poređenje a drugi, opcioni, jeste vrednost this-a. Prosleđena funkcija može da primi do tri parametra, trenutna vrednost je obavezna, a index i array su opcioni.

    const mojArray = [1,6,3];
    mojArray.findIndex(x => !(x%2))

    // Vraća 1 što je index broja 6, prvog parnog broja u Array-u

    Insertovanje elemenata u javaSkript niz .fill()

    .fill() metoda prima tri parametra, vrednost koju je potrebno ubaciti u Array, početni index, i krajnji index. Ako se izostave početni i krajnji index, svi elementi u array-u će biti zamenjeni prosleđenom vrednošću.

    arr.fill(
    "Spiridon Kopicl",
    arr.indexOf("Pera Mitić"),
    arr.indexOf("Pera Mitić") + 1
    )

    Ovako napisana naredba će zameniti prvo pojavljivanje Pere Mitića sa Spiridonom Kopiclom. Prvo, jer indexOf() pronalazi prvo pojavljivanje.

    Kopiranje dela JavaSkript niza na drugo mesto u tom istom nizu .copywithin()

    .copyWithin() metoda kopira zadati element Array-a na novu lokaciju u tom Array-u, menjajući element koji se do tada tu nalazio. Metoda ne dodaje nove elemente u Array. Prima do tri parametra, prvi je obavezan, i predstavlja index na koji želimo kopirati element, dakle index elementa koji će biti zamenjen. Druga dva parametra su opciona, i predstavljaju početni i krajnji index elemenata koje želimo kopirati. Ako se ova dva parametra izostave, početak će biti index 0, a kraj, kraj Array-a.

    const mojArray = ["Pera", "Mila", "Laza", "Živka", "Sima", "Toza", "Zaga", "Rada"]

    mojArray.copyWithin(4);

    // Vraća: [ "Pera", "Mila", "Laza", "Živka", "Pera", "Mila", "Laza", "Živka" ]
    const mojArray = ["Pera", "Mila", "Laza", "Živka", "Sima", "Toza", "Zaga", "Rada"]

    mojArray.copyWithin(4, 0, 1);

    // Vraća: [ "Pera", "Mila", "Laza", "Živka", "Pera", "Toza", "Zaga", "Rada" ]
    const mojArray = ["Pera", "Mila", "Laza", "Živka", "Sima", "Toza", "Zaga", "Rada"]

    mojArray.copyWithin(4, 0, 2);

    // Vraća: [ "Pera", "Mila", "Laza", "Živka", "Pera", "Mila", "Zaga", "Rada" ]

    Iteratori

    Iterator je interfejs koji omogućava iteraciju kroz bilo koju strukturu podataka dokle god je iterator dostupan za tu strukturu podataka. U ovom slučaju, struktura podataka je Array, i za njega je dostupan iterator.

    Iterator je objekat koji definiše sekvencu, i potencijalno return vrednost po završetku iteracije. Iterator može da bude bilo koji objekat koji implementira iterator protokol, odnosno ima next() metodu. .next() metoda vraća objekat s dve vrednosti, value koji je sledeća vrednost u sekvenci, i bulijan done, koji potvrđuje da li se stiglo do kraja iteratora. Value koji se dobija u trenutku kada done postane true, jeste rezultat iteracije.

    Najčešći iterator u JS-u jeste Array iterator, i on naprosto vraća svaki pojedinačni element array-a u sekvenci. Ovo čini da zvuči kao da nema razlike između array-a i iteratora, ali razlika je u tome što pri prolasku kroz Array, array mora biti alociran u celosti, dok iteratori konzumiraju samo koliko im je potrebno, te tako mogu i da izraze sekvence neograničene dužine, kao npr opseg intidžera od 0 do infinity broja (efektivno beskonačno).

    .keys()

    .keys() metoda kreira iterator koji sadrži samo key-eve (klučeve) array-a, iliti njegove indexe, i onda prolazi kroz svaki ključ. Metoda broji i prazna mesta, i ne prima nikakve parametre.

    const arr = ["Pera", "Mila", "Žika", "Laza"];

    const iterator = arr.keys();

    iterator.next().value; // Vraća 0
    iterator.next().value; // Vraća 1
    iterator.next().value; // Vraća 2
    iterator.next().value; // Vraća 3
    iterator.next().value; // Vraća undefined jer je done = true. Svi dalji pozivi vratiće undefined.

    .values()

    .values() radi sličnu stvar kao i keys() osim što umesto ključa, sada vraća svaku pojedinu vrednost, tako da se po njima može iterirati:

    const arr = ["Pera", "Mila", "Žika", "Laza"];

    const iterator = arr.values();

    iterator.next().value; // Vraća "Pera"
    iterator.next().value; // Vraća "Mila"
    iterator.next().value; // Vraća "Žika"
    iterator.next().value; // Vraća "Laza"
    iterator.next().value; // Vraća undefined jer je done = true. Svi dalji pozivi vratiće undefined.

    .entries()

    .entries() kombinuje keys i values, i vraća oboje kao novi array:

    const arr = ["Pera", "Mila", "Žika", "Laza"];

    const iterator = arr.entries();

    iterator.next().value; // Vraća [ 0, "Pera" ]
    iterator.next().value; // Vraća [ 1, "Mila" ]
    iterator.next().value; // Vraća [ 2, "Žika" ]
    iterator.next().value; // Vraća [ 3, "Laza" ]
    iterator.next().value; // Vraća undefined jer je done = true. Svi dalji pozivi vratiće undefined.
