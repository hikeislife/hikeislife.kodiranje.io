    Provera da li JavaSkript niz sadrži traženi element .some()

    some() prima kao argument funkciju, i onda proverava da li array sadrži element, koji zadovoljava uslov iz prosleđene funkcije. Metoda proverava članove niza, jedan po jedan, dok ne naiđe na prvi koji zadovoljava traženi uslov. Tada vraća true, i prestaje dalje da proverava. Ako ne nađe element koji zadovljava uslov, nastavlja da proverava sve dok ne stigne do kraja Array-a.

    Ova metoda će vratiti false za svaki uslov koji se proverava nad praznim Array-om.

    const mojArray = [1, 2, 4, 5];

    // Funkcija po kojoj vršimo proveru:
    const jeParanBroj = element => {
    return !(element % 2);
    };

    console.log(mojArray.some(jeParanBroj));

    // Vratiće true, jer mojArray sadrži parne brojeve

    O funkcijama sa strelicama će biti više reči u odeljku o funkcijama.

    Funkcija koju prosleđujemo some() metodi, može da bude i lambda funkcija:

    mojArray.some(x => !(x %2));

    Ako želimo da proverimo da li Array sadrži neku zadatu vrednost:

    const zadataVrednost = 3;

    mojArray.some(x => x === zadataVrednost);
    Kako je some() dodata javascript standard tek od ES3, moguće je da je neki brauzeri neće prepoznati. U tu svrhu se može koristiti sledeći polifil:
    if (!Array.prototype.some) {
    Array.prototype.some = function(fun, thisArg) {
    'use strict';

    if (this == null) {
    throw new TypeError('Array.prototype.some called on null or undefined');
    }

    if (typeof fun !== 'function') {
    throw new TypeError();
    }

    var t = Object(this);
    var len = t.length >>> 0;
    for (var i = 0; i < len; i++) {
    if (i in t && fun.call(thisArg, t[i], i, t)) {
    return true;
    }
    }

    return false;
    };
    }

    Pronalaženje indeksa traženog elementa u JavaSkript nizu .indexOf() i .lastIndexOf()

    Ako nam je potrebno da odredimo položaj nekog elementa u Array-u, koristićemo .indexOf(X) koji vraća index prvog pojavljivanja elementa, ili .lastIndexOf(X) koja vraća index poslednjeg pojavljivanja u Array-u. Kao parametar (X), prosleđujemo element čiji index tražimo u Array-u.

    Pored elementa koji tražimo, možemo proslediti i opcioni parametar, koji bi predstavljao početnu poziciju pretrage. Tako:

    mojArray.indexOf("traženi element", 10);

    // Pretražuje Array mojArray za "traženi element" počev od indexa 10

    Ako se startna pozicija ne navede, indexOf pretražuje od nultog elementa, pa sve do kraja Array-a, a lastIndexOf od poslednjeg do nultog elementa.

    .indexOf() vraća samo index prvog pojavljivanja elementa u Array-u, i analogno, lastIndexOf vraća samo index poslednjeg pojavljivanja.

    Ukoliko se traženi element ne nađe, metoda će vratiti -1

    Programeri su nekada voleli da koriste .indexOf() metodu da utvrde da li je neki element uopšte prisutan u Array-u ili ne. Ako array sadrži traženi element, indexOf vraća njegov index. Ako array ne sadrži ovaj element, indexOf vraća -1. Ako se na indexOf() primeni tilda, bitski operator, postiže se da upit vraća 0 kada element nije prisutan u array-u, a nula evaluira u bulijan false
    if (~mojArray.indexOf(element)) {
    // Uradi nešto
    }
    Ovaj uslov će vraćati true za svaki element koji jeste u Array-u a false ako element nije u Array-u.

    Ovaj hak je prevaziđen danas pošto je ES7 uveo .includes() metodu, ali moguće da ćete se susretati s njim u praksi.

    Filtriranje elemenata JavaSkript niza prema zadatom uslovu .filter()

    .filter() metoda je u neku ruku slična .some() metodi, u tom smislu da obe metode primaju funkciju, i onda proveravaju svaki član array-a, sa uslovom koji ta funkcija postavlja. Razlika je u tome što .some() vraća true ili false, u zavisnosti od toga da li je pronašla makar jedan element koji zadovoljava uslov. Filter umesto toga, kreira novi Array, u koga smešta sve elemente koji zadovoljavaju uslov iz funkcije.

    Filter prima jedan parametar obavezno (funkcija po kojoj se filtrira) i drugi opcioni parametar može biti vrednost this-a. Ako se vrednost this-a ne zada, biće uzet globalni this.

    Komparativna funkcija može da primi do tri parametra, ali samo prvi je obavezan, i predstavlja trenutni element Array-a (x u donjem primeru). Druga dva parametra bi bili index trenutnog elementa, i array koji se filtrira.

    const mojArray = [1, 2, 4, 5];
    const parniArray = mojArray.filter(x => !(x % 2));

    // Vratiće [ 2, 4 ]

    Svođenje JavaSkript niza na jedinstvenu vrednost .reduce() i .reduceRight()

    .reduce() metoda svodi čitav Array na jednu vrednost, tako što primenjuje zadatu funkciju na svakom članu Arraya, i čuva rezultat u "akumulatoru", "totalu" ili "rezultatu". Reduce prolazi kroz Array s leva u desno, a reduceRight s desna u levo.

    Reduce može da primi do dva parametra. Prvi je obavezan, i predstavlja funkciju koja se primenjuje nad svim članovima Array-a. Drugi parametar je opcion, i predstavlja početnu vrednost rezultata.

    Funkcija koja se prosleđuje može da primi do 4 parametra. Prva dva su obavezna. Prvi predstavlja trenutnu vrednost rezultata redukovanja Arraya. To je vrednost koja će biti vraćena kada se prolaz završi. Drugi obavezan parametar je trentni element niza nad kojim se proračun vrši. Druga dva parametra su opciona, i predstavljaju trenutni index, i sam array nad kojim se vrši redukovanje.

    const mojArray = [2, 5, 8, 17, 13, 98, 55, 48, 19, 3];

    function ljutiGusari(rezultat, trenutni) {
    return rezultat + trenutni;
    }

    mojArray.reduce(ljutiGusari);

    // Vraća 268, zbir svih članova Array-a

    Mapiranje JavaSkript niza .map()

    .map() kreira novi array, gde je svaki član array-a rezultat primene zadate funkcije na svaki član polaznog array-a. Prima dva argumenta, prvi je obavezan i to je funkcija koja definiše operaciju koju je potrebno izvršiti nad svakim članom polaznog array-a, a drugi parametar je opcion, i predstavlja vrednost this-a. Sama funkcija prima tri parametra. Prvi je obavezan i predstavlja trenutni član Array-a, dok su druga dva opcioni, i predstavljaju index trenutnog člana Arraya, i sam Array.

    const mojArray = [4,9,16,25];

    function korenovanje(trenutni) {
    return Math.sqrt(trenutni);
    }

    const koreni = mojArray.map(korenovanje);

    // Vraća array: [ 2, 3, 4, 5 ], gde je svaki član koren odgovarajućeg člana polaznog Array-a

    Primena funkcije na svaki član JavaSkript niza .forEach()

    .forEach() poziva prosleđenu funkciju nad svakim članom array-a. Metoda vraća undefined.

    Metoda prima do dva parametra, prvi je obavezan, i to je funkcija koju je potrebno primeniti, a drugi je vrednost this-a i opcionog je karaktera. Prosleđena funkcija prima do tri parametra, prvi obavezan parametar je trenutni element, a druga dva parametra su opcioni i to su index trenutnog elementa i array s kojim se radi.

    Provera da li svaki element JavaSkript niza zadovoljava uslov .every()

    .every(), slično some() metodi, prima funkciju, i onda proverava da li elementi Array-a zadovoljavaju uslov iz prosleđene funkcije. Metoda završava čim nađe element koji ne zadovoljava uslov, vraćajući false. Ako metoda stigne do kraja Array-a i svi elementi zadovolje zadati uslov, metoda vraća true.

    Provera da li je neka vrednost JavaSkript niz .isArray()

    Pošto typeOf arr vraća objekat, u ES5 je uvedena metoda isArray, koja vraća bulijan vrednost u zavisnosti od toga da li je data varijabla Array ili ne.

    Array.isArray(mojArray);

    Treba napomenuti da stariji brauzeri ne prepoznaju ES5 sintaksu. Da bi se ostvarila isArray funkcionalnost u starijim brauzerima, može se kreirati verzija ove metode, na sledeći način:

    function isArray(x) {
    return x.constructor.toString().indexOf("Array") > -1;
    }

    isArray(mojArray);

    Ova funkcija proverava da li konstruktor naše varijable u svom nazivu sadrži reč Array.

    Takođe se može utvrditi da je neka varijabla Array ili ne, korišćenjem operatora instanceOf

    mojArray instanceOf Array;

    Koja vraća bulijan vrednost.
