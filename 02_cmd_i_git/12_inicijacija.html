Git inicijacija: $ git init

Prvo, prevezite git u folder koji želite da pratite. Ako se ne sećate kako ovo da uradite, pogledajte ovde
MINGW32:~

$ git init
Initialized empty Git repository in /MojFolder/.git/

$_

Ono što smo ovim rekli Git-u je da tretira ovaj folder, MojFolder, kao novi Git repozitorijum, i da prati sve fajlove unutar njega. Git nas u sledećem redu obaveštava da je inicilizirao ovaj folder, i da je kreirao dodatni folder s imenom .git. U ovom folderu, on će čuvati sve izmene koje prati.

Nemojte brisati .git folder
Ovaj korak ćemo vršiti samo jednom po projektu. Od sada pa na dalje, git monitoriše ovaj folder, i jedini razlog da ponovo koristimo ovu komandu, jeste kad počinjemo neki drugi projekat, u nekom drugom folderu.

Dakle, .git folder ne treba da dirate, jedino možete da čačkate po config fajlu, ako hoćete da menjate podešavanja za dati projekat.

Svaki sledeći fajl ili folder, koji dodamo u ovaj naš folder, git će registrovati.
