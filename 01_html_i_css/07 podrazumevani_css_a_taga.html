CSS <a> taga

<a> tag takođe ima podrazumevani stil, koji sav linkovan text čini plavim i podvučenim. Ako kliknete na link, i zadržite ga tako, linkovan text će postati crven. Ako ste već posetili neki link, on će nakon toga biti ljubičast. Sva ova stanja linkova se mogu izmeniti putem CSS-a.

Pseudo klase a taga

Već smo pomenuli jednu pseudo klasu, :focus. Rekli smo da pseudoklase definišu izgled elementa u nekom specijalnom stanju. <a> tag ima nekoliko specijalnih stanja, odnosno nekoliko pseudo klasa:

  • :link
  • :visited
  • :active
  • :hover

Ako ste pomislili da se sve pseudo klase pišu s : ispred imena klase, bez razmaka, u pravu ste. Po ovome se razlikuju od istalih deklaracija.

Zapazite i da se pseudo klase obavezno pišu uz element čije stanje opisuju, ne možete definisati stanje elementa ako ne naglasite o kom elementu je reč. Pseudo klase pišemo bez razmaka s elementom na koji se odnose.

Možete zapaziti i da ako želimo dati većem broju različitih elemenata isti stil, možemo sve te elemente popisati jedan za drugim, odvojene zarezom. Zarez je bitan, i bez njega bi deklaracija značila nešto drugo.

Takođe možete primetiti, i da je moguće definisati dve pseudo klase zajedno. Ovo znači da se taj css stil, primenjuje ako je element u oba stanja u isto vreme.

pseudo klasa :link

:link pseudo element definiše stil linka, pre nego što je posetilac posetio, kliknuo, ili dok mu kursor nije iznad ovog elementa. Samim tim, :link ne definiše ništa dodatno u odnosu na a {} selektor. A kako se pri učitavanju stranice svi linkovi otvaraju u ovom stanju, i onda se primenjuju drugi stilovi u zavisnosti od toga šta korisnik radi, to se ova pseudoklasa uvek definiše pre ostalih. Ako bi bila definisana nakon ostalih a:pseudoklasa, onda se ostale ne bi ni primenjivale. Sećate se, roze potkošulja, šaren džemper?
