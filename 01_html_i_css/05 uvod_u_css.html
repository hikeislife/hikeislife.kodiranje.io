Border CSS svojstvo

U odeljku "prvi sajt" ste videli kako en: Inlineunizano ubaciti CSS kod u HTML. Sada ćemo videti drugi način da se CSS uvede u HTML, i njega nazivamo en: Embeddedusađen ili en: Internalunutrašnji CSS.

Zapazite da se unutrašnji CSS stavlja unutar <style> HTML tagova, koji se postavljaju unutar <head> tagova.

Dok je unizani CSS omražen naširoko, i opet ga svi koriste (unizani CSS je najlakši način da naterate neki HTML element da izgleda kako vi želite, čak i kada mu CSS s nekog drugog mesta, govori nešto drugo), unutrašnji CSS ima svoje ozbiljne prednosti i mane. Naime, za razliku od direktnog CSS-a, unutrašnji CSS je mnogo lakši za editovanje. A za razliku od en: Externalspoljnog CSS-a (koga ćemo obraditi kasnije), unutrašnji CSS se učitava zajedno sa HTML-om, dakle ne zahteva dodatni poziv ka serveru, i naknadni download. Iz ovog razloga unutrašnji CSS znatno doprinosi brzini pravilnog prvobitnog učitavanja strane, što je od posebnog značaja kad se sajt učitava sa mobilnog uređaja, i na šta google jako obraća pažnju kad rangira sajtove u rezultatima.

Česta je praksa da se deo CSS-a, koji se odnosi na tzv "sr lit: Iznad prevojaabove the fold" deo sajta, koji odmah postaje vidljiv, pre nego što korisnik počne da skroluje, uvodi kao unutrašnji CSS, dok se sav ostali CSS, odvaja u zaseban fajl (spoljni CSS).
Glavna mana unutrašnjeg CSS-a jeste da je unutar HTML fajla, pa je opet u slučaju kasnijih izmena, CSS potrebno tražiti na više različitih mesta.

CSS border svojstvo

border CSS svojstvo je jako korisno u ovoj ranoj fazi, zato što HTML elementu daje okvir, koji je jasno uočljiv u brauzeru, pa je lako zapaziti kako se pojedini HTML elementi ponašaju, gde počinju, a gde se završavaju.

CSS će biti primenjen na onaj HTML element koji je zadovoljen selektorom. U ovom primeru su navedena tri selektora, *, body, i body *.

* {} znači da će stil naveden u {} biti primenjen na ceo HTML dokument (osim ako neki specifičniji selektor kasnije, ne uvede drugačiji stil, kao u našem primeru). Ako ovaj kod sačuvate u .html fajl, i otvorite u brauzeru, videćete da je sveukupna sadržina stranice uokvirena crnim ramom, en: Solidpune linije, debljine 1px. Ako ne bismo imali druga dva selektora (oba specifičnija od * {}), svaki element na strani bi bio uokviren istim ramom.

Drugi navedeni selektor u primeru je "body" i odnosi se <body>...</body> tag. Sam body {} selektor je strogo specifičan, i odnosi se isključivo na body HTML tag, ali ne i na elemente koji se nalaze u njemu. Svaki HTML element možete ovako selektovati CSS-om, navedete naziv HTML taga, bez < i > zagrada, i specifičnost će uvek biti ista, s tom razlikom da možete imati samo jedan body tag po strani, i neograničeno mnogo <p> tagova. Selektor p {} bi primenio stil na sve p tagove, osim ako naravno kasnije ne bismo naveli dodatne stilove za samo neke p tagove.

Ovakva CSS deklaracija znači da će body element imati crveni en: Dottedtačkasti ram, debljine 3px.

Treći navedeni selektor je "body *". Ovaj selektor znači da će se navedeni stil primeniti na sve elemente, unutar body elementa. Pošto se unutar <body>...</body> taga nalazi vidljivi deo HTML stranice, to je razlika između ove i naše prve * {} deklaracije vrlo mala, a pošto je body * {} selektor specifičniji od * {}, to će i sav sadržaj na toj strani, a unutar <body> imati zeleni en: Dashedcrtičasti okvir, debljine 5px.

Border ne mora da se definiše za sve četiri strane HTML elementa, kao što smo to uradili u gornjem primeru. Takođe, širina, stil i boja bordera, svi mogu da se definišu ponaosob.

border-width ili širina okvira

border-width: vrednost; je CSS svojstvo kojim se definiše širina (debljina) okvira nekog elementa. Može imati 1 od 4 i to:

  • "thin" tanka
  • "medium" ili srednja, i ovo je podrazumevana vrednost ako se svojstvo ne navede
  • "thick" ili debela, i
  • brojna vrednost, prikazana je širina od 1px koja je u hromu šira od thin, a tanja od medium, dok na Firefox-u izgleda isto kao "thin"

primer:

Ovako definisano svojstvo znači da će okir sa sve 4 strane biti debela linija. Dakle, ako navedemo samo jednu vrendost, ona se odnosi na sve 4 ivice.

Primer

Ovako definisan border-width znači da će gornji i donji ram biti debeli, a levi i desni tanki. Dakle ako navedemo dve vrednosti, prva se uvek odnosi na gorni i donji ram, a druga na levi i desni.

Primer

Kada navedemo tri vrednosti, prva se odnosi na gornji ram, koji će u ovom primeru biti debeo, druga vrednost, medium, se odnosi na levi i desni ram, koji će u ovom slučaju biti srednje debljine, a treća vrednost, thin, se odnosi na donji ram, koji će biti tanak.

Primer

Konačno ako navedemo sve četiri vrednosti;

Tada se vrednosti odnose na strane u smeru kazaljke na satu, počev odozgo, dakle: gornja (20px), desna (10px), donja (5px), leva (0).

Primer

Zapazite da se vrednosti u sva tri slučaja odvajaju praznim mestom, i da ako je vrednost 0, ne pišemo jedinice (px). Jedinice kada se pišu, pišu se bez razmaka posle brojne vrednosti. Bilo koja omaška u formatu, rezultiraće time da se border ne pojavi na elementu. Dok je HTML vrlo fleksibilan sa greškama, CSS je izuzetno sitničav, jedan izostavljen ; i sve stade, on nema pojma šta ste vi mislili, on samo zna šta ste napisali.

border-color ili boja okvira

I u ovom slučaju se može navesti od jedne do 4 vrednosti, na isti način kao u prethodna dva slučaja. Boja može da se definiše na sve načine koji su uobičajeni pri definisanju boja, o čemu će više reči biti kasnije, u poglavlju "boje"

Ukoliko se boja ne navede posebno, okvir će biti iste boje kao tekst

border-style ili stil bordera

Stil bordera se takođe može definisati sa 1 - 4 vrednosti, kao u slučaju border-width svojstva. U ovom slučaju te vrednosti mogu biti:

  • "none" je podrazumevana vrednost, nema okvira
  • "hidden" takođe nema bordera, a koristi se kad imamo konflikt u definiciji bordera kod tabela
  • "dotted" tačkasti okvir
  • "dashed" crtičasti okvir
  • "solid" okvir je puna linija
  • "double" dupla linija
  • "groove" definiše 3d žljeb čiji efekat zavisi od boje okvira - vi definišete samo jednu boju, brauzer izračunava drugu.
  • "ridge" definiše 3d greben čiji efekat zavisi od boje okvira
  • "inset" definiše 3d uleglunće čiji efekat zavisi od boje okvira
  • "outset" definiše 3d ispupčenje čiji efekat zavisi od boje okvira
  • "initial" podešava vrednost na početnu
  • "inherit" naslediće okvir od elementa u kome se nalazi
Od tri karakteristike border svojstva, border-style je jedina koja mora da se navede. Podrazumevana vrednost za border-style je "none", tako da ako se ova karakteristika ne navede, neće ni biti bordera. Ako je border-style jedina navedena, okvir će biti medium širine, i imaće istu boju kao i uokvireni tekst.
