CSS <html> taga

CSS display svojstvo

Svi HTML elementi se mogu posmatrati kao kućice unutar kojih se nalazi neki tekst. Kao takvi, tagovi se mogu podeliti u dve grupe. One kod kojih je podrazumevana vrednost sr: Prikazdisplay-a "block", i one kod kojih je ova vrednost "inline"

Tagovi koji se prikazuju kao blokovi, zauzimaju celu širinu stranice, tako da se svaki sledeći blok može nalaziti samo ispod njih, i nikako u nastavku. Primer su <body>, <p>, i svi <h_> tagovi.

Tagovi koji se prikazuju kao inline, se prikazuju u nastavku teksta, a primer je <a> tag. Karakteristika inline elemenata je da im je nemoguće definisati visinu ili dužinu. Oni su uvek dugački onoliko koliko slova u njima zahtevaju, i visoki onoliko redova koliko obuhvataju. Usled ovoga ni se lepi uklapaju u okolni tekst.

Svi HTML elementi, po defaultu, imaju ili block, ili inline vrednosti za display svojstvo. Međutim vremenom su uvedene i mnoge druge vrednosti za različite svrhe i to:

  • "sr: Unizanoinline" - prikazuje element u nastavku.
  • "sr: Blokoviblock" - element će započeti u novom redu, i zauzeće celu raspoloživu širinu ekrana
  • "sr: Unizani blokoviinline-block" - element će i dalje biti prikazan u nizu, ali sada mu se mogu definisati visina i širina
  • "sr: Ništanone" - element i njegov sadržaj, neće biti prikazani (ovo je korisno kod dizajna menija u kome se opcije pojavljuju tek kada kliknete na neki element menija, ili kod dizajna koji se razlikuje za mobilne i desktp platforme)
  • "sr: Sadržinacontents" - element neće biti prikazan, ali njegov sadržaj hoće, tako da se ovim remeti struktura DOM - Document Object Model; sr: Model dokumenta kao skupa objekataHTML stabla - ne radi u IE i Edge brauzeru. Efekat je takav kao da je sav sadržaj ovakvog elementa, zbrljan u <p> tag. U slučaju tabela, u niz <p> tagova.
  • "sr: Početnoinitial" - resetuje display na podrazumevanu vrednost
  • "sr: Naslednoinherit" - naslediće svojstvo od roditeljskog elementa u DOM - Document Object Model; sr: Model dokumenta kao skupa objekataHTML stablu
  • "odnosi se na grid boxgrid", "odnosi se na grid boxinline-grid" - biće obrađeni kad budemo radili CSS grid
  • "odnosi se na flex boxflex", "odnosi se na flex boxinline-flex" - biće obrađeni kad budemo radili CSS flex (u Safari-ju zahtevaju -webkit- prefiks
  • "table", "inline-table", "table-caption", "table-column-group", "table-header-group", "table-footer-group", "table-row-group", "table-cell", "table-column", "table-row" - biće obrađeni kad budemo radili tabele
  • "sr: Stavka na listilist-item" - prikazuje element kao <li> element (vidi liste)
  • "sr: Utrčavajućarun-in" - element će biti prikazan ili kao blok, ili u nizu, zavisno od konteksta

:focus selektor

:focus, je jedna od pseudo klasa u CSS. Pseudo klase se koriste da definišu specijalno stanje elementa. Ovaj selektor se koristi kod elemenata koji dozvoljavaju unos preko tastature. Kada se kursor nalazi u takvim elementima, smatra se da imaju fokus. U opštem slučaju, ovakvi elementi dobiju "sr: Okvirna linija, javlja se oko border svojstvaoutline" kad su u fokusu. Uobičajeno je da se outline onda ovakvim selektorom ukloni, ili izmeni u nešto prikladnije za dati sajt. Druga svojstva se mogu takođe precizirati. Npr, možete imati polje za pretragu, koje bi normalno bilo usko, ali bi se proširilo kada biste kliknuli da ukucate traženi pojam. O outline svojstvu će kasnije biti više reči.

CSS margin svojstvo

Margina predstavlja deo HTML stanice koji se nalazi neposredno izvan kućice nekog taga. Svi HTML tagovi su okruženi marginom. Ako se stavi da je margin: 0; onda će ovaj element leći neposredno pored prethodnog elementa, dok margin: 8px; znači da će postojati 8 piksela razmaka između body taga, i njegove okoline (a za <body> to je <html> tag). Dakle, ako ovu vrednost ne promenimo, telo našeg sajta će imati marginu od 8 piksela, koja će se manifestovati odbojnikom oko celog sadržaja sajta. Ako bismo body tagu dodali en: Borderokvir jasno bismo videli da body tag odstupa od ivice ekrana za 8px.

font-size svojstvo

font-size definiše veličinu slova. Može se definisati u različitim jedinicama, najintuitivnija je svakako px.

em jedinica je relativna u odnosu na veličinu slova. 1em, je veličina jednog slova. Ako je veličina slova 16px, 1em je jednak 16px, .5em je jednaka 8px, 2em je 32px itd.

em jedinica potiče iz tipografije, gde je em značilo: "širina slova m", a ex: "visina slova x". em-ovi su se koristili da definišu širine, a ex-ovi da definišu visine. U web dizajnu, ex se vrlo retko koristi. em se koristi i za visine i za širine, i definiše se kao vrednost relativna u odnosu na veličinu slova, svih slova u elementu, radije nego kao širina slova m.

Kada je kao u ovom sl. veličina slova definisana u em-ovima, to znači da je ona relativna u odnosu na veličinu slova definisanu u root elementu.
